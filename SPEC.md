# AI小说创作网站 - 规格说明书

## 项目概述
- **项目名称**: 墨韵AI (MoYun AI)
- **类型**: AI辅助小说创作Web应用
- **核心功能**: 通过AI辅助设置主题、构建大纲、创作内容
- **目标用户**: 小说作者、写作爱好者

## 功能列表

### 1. 主题设置
- 预设主题类型选择（言情、玄幻、悬疑、科幻、武侠、悬疑、都市、历史）
- 自定义主题输入
- AI分析并提供主题细化建议
- 保存主题设定

### 2. 环境背景设置
- 世界观设定
- 时代背景选择
- 社会环境描述
- 地理/空间设定
- 规则设定（魔法体系、科技水平等）

### 3. 大纲管理
- 创建小说大纲（章节结构）
- 添加/编辑/删除章节
- 拖拽调整章节顺序
- AI辅助生成章节建议

### 4. 内容创作
- 逐章节撰写内容
- AI续写/润色功能
- 字数统计
- 历史版本记录

### 5. AI对话功能
- 内置AI助手提供写作建议
- 角色/情节/世界观咨询
- 根据当前小说类型调整回复风格

### 6. 主题专属提示词系统
不同主题使用不同的AI系统提示词：
- **言情**: 注重情感细腻描写、人物心理刻画
- **玄幻**: 世界观构建、魔法体系、升级体系
- **悬疑**: 悬念铺设、线索埋设、节奏把控
- **科幻**: 科技设定、逻辑自洽、未来推演
- **武侠**: 江湖规矩、武功招式、门派纷争
- **都市**: 现代生活、人际关系、社会现实
- **历史**: 时代还原、历史细节、人物风貌

## 技术方案
- **前端**: HTML5 + CSS3 + Vanilla JavaScript
- **后端**: Python Flask
- **AI接口**: Claude API (可选本地模拟)

## 视觉设计
- 东方古典文学风格
- 配色: 米色底(#F5F0E6)、墨色字(#2C2C2C)、朱红点缀(#C54B4B)
- 字体: 思源宋体/Noto Serif SC
- 布局: 左右分栏（设置区/创作区）
