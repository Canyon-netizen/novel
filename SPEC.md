# AI小说创作网站 - 规格说明书

## 项目概述
- **项目名称**: 墨韵AI (MoYun AI)
- **类型**: AI辅助小说创作Web应用
- **核心功能**: 通过AI辅助设置主题、构建大纲、创作内容
- **目标用户**: 小说作者、写作爱好者

## 技术架构
- **前端**: HTML5 + CSS3 + Vanilla JavaScript
- **后端**: Python Flask (可选，本地模拟模式可用)
- **数据存储**: LocalStorage + IndexedDB
- **AI接口**: 支持多Provider (Anthropic Claude, OpenAI GPT, DeepSeek, MiniMax, Kimi, GLM, 自定义)

## 页面结构

### 1. 首页 (index.html)
- 欢迎语展示（根据时间自动问候）
- 统计数据面板（小说项目数、总章节数、总字数、总角色数）
- 小说项目列表（网格/列表视图切换）
- 搜索和筛选功能（按类型、按时间排序）
- 新建小说入口

### 2. 创建页 (create.html)
- 小说基本信息配置（名称、主角、创作方向）
- 小说类型选择（玄幻、仙侠、都市、科幻、历史、言情、悬疑、武侠、游戏、末世）
- 叙事套路标签（系统流、升级流、重生、穿越等）
- 受众定位（男频/女频、青年向/青少年）
- 篇幅字数配置（滑块 + 预设按钮）
- 情节结构模式（线性、环形、倒叙、多线、网状）
- 模板市场（预设模板 + 自定义模板管理）
- AI辅助命名功能

### 3. 编辑页 (editor.html)
- 章节侧边栏管理
- 富文本编辑区
- AI功能工具栏（续写、润色、建议）
- 字数统计
- 自动保存
- 项目导出（Markdown格式）

## 功能模块

### 1. 主题设置
- 预设主题类型选择
- 自定义主题输入
- AI分析并提供主题细化建议

### 2. 环境背景设置
- 世界观设定
- 时代背景选择
- 社会环境描述
- 地理/空间设定
- 规则设定（魔法体系、科技水平等）

### 3. 大纲管理
- 创建小说大纲（章节结构）
- 添加/编辑/删除章节
- 拖拽调整章节顺序
- AI辅助生成章节建议

### 4. 内容创作
- 逐章节撰写内容
- AI续写/润色功能
- 字数统计
- 历史版本记录
- Markdown导出

### 5. AI对话功能
- 内置AI助手提供写作建议
- 角色/情节/世界观咨询
- 根据当前小说类型调整回复风格
- 多API Provider支持
- 本地模拟模式（无需API Key）

### 6. 主题专属提示词系统
- **言情**: 注重情感细腻描写、人物心理刻画
- **玄幻**: 世界观构建、魔法体系、升级体系
- **悬疑**: 悬念铺设、线索埋设、节奏把控
- **科幻**: 科技设定、逻辑自洽、未来推演
- **武侠**: 江湖规矩、武功招式、门派纷争
- **都市**: 现代生活、人际关系、社会现实
- **历史**: 时代还原、历史细节、人物风貌

### 7. 数据同步
- LocalStorage本地持久化
- GitHub Gist云端同步（可选）

## 数据结构

### 项目 (Project)
```javascript
{
  title: string,           // 小说标题
  type: string,            // 小说类型
  description: string,     // 简介
  protagonist: string,      // 主角名
  tropes: string[],        // 叙事套路
  audience: string,        // 受众定位
  wordCount: number,        // 预计字数
  plotStructure: string,   // 情节结构
  chapters: Chapter[],     // 章节列表
  characters: Character[],  // 角色列表
  createdAt: string        // 创建时间
}
```

### 章节 (Chapter)
```javascript
{
  title: string,           // 章节标题
  summary: string,         // 章节简介
  content: string          // 章节内容
}
```
